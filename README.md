# Rescue-AI

RescueAI is a multi-agent autonomous disaster response system that uses AI agents, data analytics  to analyze disaster situations and make real-time decisions on priority zones, resource allocation, and emergency alerts. It turns hours of manual disaster management work into 3 seconds of automated AI-driven action.
